# Personal Portfolio Website


A modern, responsive portfolio website showcasing my skills, projects, and professional experience as a Fullstack Engineer. Built with modern HTML, CSS, and JavaScript, it integrates animations and interactive effects for an engaging user experience.


## üåü Main Features

- **Responsive Design**: Optimized for all devices.
- **Scroll Animations**: Powered by [AOS](https://michalsnik.github.io/aos/) for smooth on-scroll effects.
- **Interactive Particle Background**: Uses [tsParticles](https://particles.js.org/) for dynamic particle effects.
- **Modern UI/UX**: Dark theme, neon green accents, glassmorphism, and custom scrollbar.
- **Complete Sections**:
  - Skills & Expertise
  - Education
  - Professional Experience
  - Project Showcase
  - Contact Information
- **Hover & Floating Effects**: Neon and animated effects on social links.


## üõ†Ô∏è Technologies & Code Structure

- **HTML5**: Main structure in `index.html`.
- **CSS3**: Modular, with structure:
  - `src/styles/main.css` (entry point, imports base, components, utilities)
  - Components: hero, about, sections, projects, footer (glassmorphism, neon accent, responsive)
  - Utilities: custom scrollbar, AOS animations
- **JavaScript**: 
  - `src/script.js` for initializing AOS and tsParticles (animations & interactive particles)
- **AOS**: Scroll animation library
- **Font Awesome**: Social icons
- **tsParticles**: Particle effects (replacement for Particles.js)

4. **Anevia Backend Model**
   - Machine learning model service for Anevia, enabling automated anemia detection from eye conjunctiva images using advanced AI algorithms.

5. **MeNovel**
